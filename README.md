# Socket.io Chat (Node.js)

Projeto simples para estudo de comunicação em tempo real usando Socket.io.

## Objetivo
Entender como funciona a comunicação entre backend e frontend com WebSockets,
registrando conexões, desconexões e troca de mensagens em tempo real.

## Tecnologias
- Node.js
- Express
- Socket.io
- Nodemon
- HTML + CSS

## Funcionalidades
- Log de usuários conectados/desconectados no servidor
- Envio de mensagens em tempo real
- Broadcast para todos os usuários conectados

## Aprendizados
- Diferença entre estrutura de pastas e rotas expostas pelo Express
- Uso do express.static para servir arquivos do frontend
- Comunicação bidirecional com Socket.io
- Importância de começar pequeno antes de escalar um projeto
